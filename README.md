# ⚙️ Automation System Simulator

## 🧠 Overview
This project demonstrates how automation systems work using rule-based logic.

It simulates how a system can take input, process it, and automatically produce decisions without human intervention.

---

## ⚙️ Features
- Multi-input processing (temperature, humidity, time)
- Rule-based intelligent decisions
- Environment simulation
- Automated system responses

---

## 🛠️ Tech Stack
- Java

---

## 🚀 Purpose
To showcase practical understanding of automation and system design.
